# Warehouse Inventory App

## Uruchomienie
1. Zainstaluj Node.js 20+.
2. W katalogu głównym uruchom:
   ```bash
   npm install
   npm run install:all
   ```
3. Skopiuj `backend/.env.example` do `backend/.env`.
4. Uruchom:
   ```bash
   npm run dev
   ```

## Test QR
- Otwórz frontend w przeglądarce na telefonie lub laptopie.
- Zezwól na dostęp do kamery.
- Zeskanuj QR produktu, wpisz ilość, zeskanuj QR regału i zapisz operację.

## CSV
- W każdej zakładce kliknij eksport CSV.
- Plik pobierze się z nagłówkami i kodowaniem UTF-8.
